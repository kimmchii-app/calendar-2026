# 我請克勞德寫得月曆設定器(只有2026年)
- claude ai協助編寫
- 中間有用Canvas API
- 完全可以用我得再自己修改
## 使用說明
- 分成左、中、右三個區塊
- 大部分按鈕自己動一、滑桿滑一滑就可以了
- 下載檔案可以存成png，一次下載12張可以
## 左區
- 選擇月份的地方，上面有出現小橘點代表那個月份有被編輯過
- 目前得月份只有2026，想要2027可以自己再請ai修改
## 中區
- 成品預覽區
- 放置圖片可以直接點選畫面中間得大空白處
## 右區
- 控制板區，可以調整大小、縮放、顏色、選幾張圖
- 目前大小有3:4、4:3、1:1、9:16、16:9，共五種
- 可選版面是左曆右圖、左圖右曆、上曆下圖、上圖下曆、滿版，共五種
- 拼貼圖片張數最多為4張，可以方格拼貼或梯形拼貼
- 文字、月曆皆可改色，字體也可以上傳自己喜歡的來用
- 節氣、節日皆可關閉
